# PeerMeet – Real-Time Video Conferencing Platform

PeerMeet is a full-stack video conferencing web application that enables users to create and join meetings instantly using a unique meeting code. It supports real-time video/audio communication, screen sharing, live chat, and meeting history.

---

##  Live Demo

* **Frontend:** https://peermeetfrontend2.onrender.com
* **Backend API:** https://peermeetbackend-d7qw.onrender.com

##  Features

*  User Authentication (Register & Login)
*  Real-Time Video Calling using WebRTC
*  Audio Mute / Unmute
*  Video On / Off
*  Screen Sharing
*  Real-Time Chat during meetings
*  Meeting History
*  Protected Routes using Authentication
*  Dynamic Meeting Links
*  Fast and Responsive UI

## Application Workflow

1. User registers or logs in.
2. Authentication token is stored locally.
3. User enters a meeting code or creates one.
4. Meeting history is saved in MongoDB.
5. WebRTC establishes peer-to-peer connections.
6. Socket.IO handles signaling.
7. Users can:

   * Toggle Video
   * Toggle Audio
   * Share Screen
   * Send Messages
8. User leaves the meeting.
